# Image_gallery

A simple and responsive Image Gallery Web Application built using HTML, CSS, and JavaScript. The gallery displays beautiful images and allows users to filter them by category such as Nature and City using interactive buttons.

🚀 Features
Responsive image gallery layout
Filter images by category
Smooth and clean user interface
Hover effects on images
Lightweight and easy to customize
Beginner-friendly project

🛠️ Technologies Used {
HTML5
CSS3
JavaScript (JS) }
